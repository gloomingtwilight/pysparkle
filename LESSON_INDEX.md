## 🟢 BEGINNER LEVEL - 12 Lessons (Foundation)

| ID | Lesson | Focus Area | Key Concepts |
|:--:|---------|-----------|--------------|

| 2 | **Variables & Data Types** | Variables | `int`, `str`, `float`, assignment |

| 4 | **Operators** | Math & Logic | `+`, `-`, `*`, `/`, `and`, `or` |
| 5 | **Conditional Statements** | Flow Control | `if`, `elif`, `else` |

| 8 | **Lists** | Data Structures | indexing, slicing, `.append()` |
| 9 | **Tuples & Sets** | Collections | immutable sequences, unique values |
| 10 | **Dictionaries** | Key-Value Pairs | keys, values, `.get()` |
| 11 | **Strings** | Text Processing | concatenation, methods |
| 12 | **🎮 Mini Project** | Synthesis | **Calculator / Guessing Game** |

### Learning Outcome
After Beginner level, you can:
- Write basic Python programs
- Handle data with variables and collections
- Use control flow for decision-making
- Organize code with functions
